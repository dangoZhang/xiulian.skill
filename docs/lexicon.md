# 修仙彩蛋表

这份表只服务修仙彩蛋模式。

使用规则：

- 默认报告先写人话，不默认切修仙。
- 用户明确要求修仙彩蛋时，先按这份表做对应词替换。
- 表中顿号、斜杠都表示“或”的关系，不表示必须同时出现。
- 第一轮替换完成后，再做第二轮改写，把文本润成修仙小说风格，同时保持原始事实、层级、短板、建议不丢失。
- 如果某个词替换后明显别扭，就保留原始 AI 术语，不强翻。

## 修仙彩蛋改写流程

当用户明确要求修仙版报告、修仙版分享卡、修仙判词时，按下面顺序处理：

1. 先生成默认人话版报告。
2. 对照本表，把原始报告中的对应词替换为修仙词。
3. 替换后再做一轮整体改写，让句子符合修仙小说风格，要求流畅、连贯、像人写的。
4. 改写时必须保留：
   - 原来的等级或境界判断
   - 原来的能力描述
   - 原来的短板
   - 原来的突破方向
   - 原来的 token / 模型 / 时间窗 / 样本规模等真实依据
5. 如果修仙写法会让信息变糊，就回退到原始 AI 术语或括号保留原词。
