# 修仙画像生成 Prompt

用于生成「修仙画像」分享图。目标是像一张可晒图的符箓卷轴，修仙味足，但信息结构必须清楚。

## 变量

- `{{title}}`：修仙画像
- `{{username}}`：用户名或道号
- `{{generated_at}}`：生成时间
- `{{realm}}`：境界，如金丹、元婴、化神
- `{{subtitle}}`：一句副题，点出此人的修行路数
- `{{summary}}`：仙途评语，2 到 4 句，写出这次画像的核心判断
- `{{growth}}`：破境之法，1 到 2 句

## Prompt

请生成一张 3:4 竖版中文海报，主题为“修仙画像”。整体风格要像赛博修仙里的高阶符箓卷轴，克制、庄重、留白得当，有纸张、金墨、朱砂、阵纹这些明确修仙元素，但不能俗艳。

画面结构：

1. 顶部中央标题必须是“{{title}}”。
2. 标题下方放一行副题：`{{subtitle}}`。
3. 中央主视觉是大号境界字：`{{realm}}`，下方小字是“当前境界”。
4. 境界区下方是第一块信息区，标题为“仙途评语”，内容是 `{{summary}}`。
5. 再下方是第二块信息区，标题为“破境之法”，内容是 `{{growth}}`。
6. 持有人 `{{username}}` 与生成时间 `{{generated_at}}` 放在整张图最底部的一行小字里，不再额外放其他信息格。

视觉要求：

- 以卷轴、符箓、金墨、朱砂、米纸、云气为主，图案只服务信息层级。
- 修仙语汇要贴近日常网文修仙设定，重点写境界、根骨、破境，不要自造门派设定和生造术语。
- 标题、境界字、正文、小字四层层级要非常清楚，适合直接分享。

排版硬约束：

- 仙途评语和破境之法两个框的位置，都必须在各自最后一行文字的下方，再额外留出一整行字高的安全距离。
- 所有文字必须清晰、可读、对齐，不能重叠，不能压框，不能超出边界。
- 不要六宫格，不要塞太多字段，不要因为装饰牺牲正文空间。

禁止项：

- 不要人物立绘、武器、火焰、满屏八卦纹、龙凤等廉价玄幻元素。
- 不要无意义图腾，不要做成小说封面或游戏 UI。
- 不要出现直白的 AI 术语，把修仙画像保持在修仙叙事里。
