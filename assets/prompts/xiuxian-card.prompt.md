# 修仙.skill 单卡生成 Prompt

用于生成 `修仙.skill` 的单张分享卡。目标是可晒图、可转发、信息一眼可读。

## 变量

- `{{title}}`：修仙.skill
- `{{username}}`：命主
- `{{generated_at}}`：起炉时间
- `{{realm}}`：境界，如 金丹、化神、大乘
- `{{rank}}`：等级，如 L4、L7、L10
- `{{ability}}`：能力描述，修仙叙事，并保留括号中的 AI 术语
- `{{verdict}}`：判词，2 到 3 句
- `{{breakthrough}}`：破境之法，1 到 2 句
- `{{model}}`：炉主模型
- `{{usage}}`：耗材，如 `538,620 token`
- `{{sample}}`：样本规模，如 `24 messages · 6 tool calls`
- `{{accent_color}}`：等级色，参考洛谷色阶

## Prompt

请生成一张 3:4 竖版中文海报，主题是“修仙.skill”。整体是赛博修仙中的高阶修为卡，适合社交传播。

画面结构：

1. 顶部是一条醒目的等级色横幅，写 `{{title}}`，副题可写“蒸馏你的 vibe coding 修为”。
2. 中央第一块是主视觉：
   - 大号境界字：`{{realm}}`
   - 旁边的大号等级：`{{rank}}`
   - 等级色必须明显，参考洛谷分级色阶。
3. 第二块是“蒸馏能力”：
   - 放 `{{ability}}`
   - 文案必须是修仙叙事
   - 括号中必须保留中英文 AI 术语
4. 同一块里继续放“判词”：
   - 内容为 `{{verdict}}`
5. 下一块是“破境之法”：
   - 内容为 `{{breakthrough}}`
6. 最底部三枚信息 chip：
   - 炉主：`{{model}}`
   - 耗材：`{{usage}}`
   - 样本：`{{sample}}`
7. 页脚小字放：
   - 命主：`{{username}}`
   - 起炉时间：`{{generated_at}}`

视觉要求：

- 修仙小说语汇要正，不要乱造门派、灵兽、法宝设定。
- 大境界字必须足够有冲击力，适合截图传播。
- 等级色需要形成鲜明记忆点。
- 整体高级、克制、干净，不要花哨游戏 UI 感。

硬约束：

- 所有文字必须完整落在信息框内部。
- 所有边框、面板、chip 的下边缘，都必须在最后一行文字的下方，并至少再留出一整行字高的安全边距。
- 小字也必须可读，不能发灰，不能糊成一团。

禁止项：

- 不要双卡，不要卷轴拼贴，不要奖杯证书感。
- 不要人物立绘、武器、龙凤、火焰、满屏符纹。
- 不要把 AI 术语翻译掉，括号中的英文术语必须保留。
