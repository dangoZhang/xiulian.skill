# AI 协作能力证书生成 Prompt

用于生成「AI 协作能力证书」分享图。目标是简洁、高级、可晒图，适合 README 预览和社交传播。

## 变量

- `{{title}}`：AI 协作能力证书
- `{{username}}`：用户名
- `{{generated_at}}`：生成时间
- `{{level}}`：证书等级，如 `L4`、`L7`、`L10`
- `{{ability}}`：一句能力描述，如“协同多个分身与工具完成任务”
- `{{verdict}}`：判词摘要，2 到 3 句，写清这次协作的核心表现
- `{{usage_meta}}`：用量摘要，如 `24 messages · 6 tool calls`
- `{{usage_text}}`：用量主值，如 `538,620 token`
- `{{theme_name}}`：主题色名，如 blue、gold、red
- `{{accent_color}}`：强调色，如 `#7EAEFF`
- `{{panel_color}}`：主面板深色，如 `#162B49`

## Prompt

请生成一张 3:4 竖版中文海报，主题为“AI 协作能力证书”。整体风格要像高质量等级认证卡，专业、克制、利落，信息密度高，但不能拥挤。

画面结构：

1. 顶部中央标题必须是“{{title}}”。
2. 标题下方只保留一行极简副题，可表达“基于真实会话稳定定级”这类含义。
3. 中央深色主面板是整张图的核心，内部只保留五层信息：
   - 超大等级：`{{level}}`
   - 小字标签：`当前等级`
   - 能力句：`{{ability}}`
   - 判词区：`{{verdict}}`
   - 底部信息盒：上行 `{{usage_meta}}`，下行 `{{usage_text}}`
4. 持有人 `{{username}}` 与生成时间 `{{generated_at}}` 放在整张图最底部的一行小字里，不额外再做独立信息框。

视觉要求：

- 采用高级证书卡视觉，背景干净，主体明确，允许少量光晕和金属描边。
- 主色使用 `{{theme_name}}` 对应的等级色，强调色使用 `{{accent_color}}`，主面板使用 `{{panel_color}}`。
- 等级必须一眼看见，能力句比判词更醒目，判词区块要稳定、整齐、容易截图分享。
- 全图不要英文装饰语，但可以保留 `messages`、`tool calls`、`token` 这类 AI 术语。

排版硬约束：

- 所有边框、面板、信息盒的下边缘，必须在最后一行文字的下方，至少再留出一整行字高的安全距离。
- 判词区和底部信息盒之间必须有明显留白，不能挤压，不能让文字贴边。
- 字号、行距、字色要统一，不能出现同一块里忽大忽小、深浅失衡、块距混乱。
- 任何文字都不能溢出框线，不能重叠，不能因为换行导致视觉重心失衡。

禁止项：

- 不要修仙、符箓、人物、奖杯、勋章、机器人头像。
- 不要堆无意义图案，不要做成后台面板、PPT、简历模板。
- 不要把 `token` 翻译成“令牌”之类的词。
