# 修仙.skill

`修仙.skill` reads real agent transcripts and distills one shareable cultivation card from your vibe coding history.

It reads Codex, Claude Code, OpenCode, OpenClaw, Cursor, and VS Code transcripts, supports latest-session, full-history, and time-window analysis, and keeps local memory for breakthrough comparison.

It outputs one unified card:

- realm
- level
- cultivation-style ability summary with AI terms
- breakthrough guidance
