# 修炼.skill

`修炼.skill` reads real agent transcripts, distills your vibe coding ability, generates one shareable card, and gives you a concrete next-cycle breakthrough plan.

It reads Codex, Claude Code, OpenCode, OpenClaw, Cursor, and VS Code transcripts, supports latest-session, full-history, and time-window analysis, and keeps local memory for breakthrough comparison.

It outputs one unified card and one coaching layer:

- realm
- level
- vibe coding ability summary
- breakthrough guidance
- direct prompts for the next cycle
